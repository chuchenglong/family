v0.1（初始版本）
---------------------------------------------------------
功能内容：
1.登录/退出，基础功能实现。
2.注册模块，基础功能实现。
3.导航和主页，前端主架子搭建。
4.账户模块测试案例。

前端主要技术点：
1.bootstrap3.3.7及相关组件。
2.自己的mc-x相关封装文件，mc-ajax.js/mc-alert.js/mc-constant.js/mc-core.js/mc-window.js。

后端主要技术点：
1.java/maven项目。
2.使用springboot。
3.加入AOP，AspectController和AspectService，对接口异常统一规范处理。
4.ConfigDataSource支持多数据源连接。
5.两个mybatis配置，使用一个，预留一个。
6.springmvc的配置预留。
7.加入请求过滤器，其中解决跨域。
8.封装结果格式化类ManagerResult，统一管理结果。
9.BaseController的请求/返回统一封装。
10.RESTFUL风格。
11.mysql数据库。
